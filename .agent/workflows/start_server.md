// turbo
1. Start the local HTTP server for the wiki on port 8080:
```
python3 -m http.server 8080
```
Run this from the `/Users/lap/23-steps/wiki` directory.

// turbo
2. Open the wiki in the default browser:
```
open http://localhost:8080/items.html
```

To stop the server, use /close_server or press Ctrl+C in the terminal.
